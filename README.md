# Web-Home 👁
*Проект с открытым исходным кодом, предназначенный для безопасного общения в локальной сети.*
# Описание
**Доступный каждому мессенджер, который ты можешь запустить за 2 нажатия.**

**Функции:**
- **Доступ к чату по логину и паролю.**
- **AES шифрование.**
- **WEB интерфейс.**
- **Личные + открытые чаты.**
